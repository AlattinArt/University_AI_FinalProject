# 树院智管 · 校园物资智能交互系统

> 本项目是浙江树人学院《人工智能》课程大作业

## 项目简介

基于 HTML5 + Tailwind CSS + 原生 JavaScript + ECharts 的智慧校园物资管理交互式前端系统，浏览器直接运行，无需客户端。

**主题赛道**：智慧校园（浙江树人学院实训器材、社团物资、办公耗材资产管理）

**核心AI交互能力**：对接云端大模型API（通义千问），实现智能问答、校园文书生成、耗材数据分析三大AI功能

## 技术栈

- **HTML5 + Tailwind CSS** — 页面结构与响应式样式
- **原生 JavaScript** — 全部交互逻辑、数据计算、DOM操作
- **ECharts** — 可视化图表（消耗趋势折线图、经费结构环形图）
- **Node.js 原生 http** — 零依赖中转层（仅用于隔离大模型API密钥）
- **Open Code** — 全程主导代码审查、逻辑优化、功能迭代

## 核心功能

### 场景1：物资申领归还管理页
- 可视化数据看板（今日申领/归还、库存总量、环比增减）
- 物资占比环形图
- 多条件联动筛选（库房、学院、物资类型、日期）
- 弹窗表单新增申领（含前端校验）
- 表格分页 + 自定义每页条数
- 单据详情查看弹窗
- A4打印预览 + Excel批量导出

### 场景2：耗材经费数据分析报表页
- 时间区间 + 库房联动筛选
- 消耗趋势折线图（按天/按月切换，悬浮明细）
- 经费结构环形图（图例交互隐藏/展示）
- 物资排行表格（月度消耗金额降序）
- 汇总统计（总耗材、总经费、平均损耗率）
- 一键导出《浙江树人学院月度耗材分析报告》Excel

### 场景3：AI智能助手对话页
- 校园物资智能问答（Prompt限定树人校园相关内容）
- 标准化校园文书自动生成（申请单、借用报告、耗材总结、报修单）
- 耗材Excel上传智能分析（浪费分析、节约方案、采购建议）
- 流式打字动画模拟实时对话
- 历史对话本地存储（刷新不丢失）
- 无关问题前端拦截

## 快速开始

### 1. 浏览器直接运行（核心功能）

用 Chrome/Edge 浏览器直接打开以下HTML文件：
- `login.html` — 系统登录
- `stock-apply.html` — 物资申领管理
- `report-data.html` — 数据分析报表
- `ai-assistant.html` — AI智能助手

> 注意：AI对话页面需要启动Node中转层才能调用大模型API

### 2. 启动AI中转服务（可选）

```bash
# 配置环境变量（复制 .env.example 并填入真实API Key）
copy .env.example .env

# 启动中转服务（默认3000端口，零依赖，无需npm install）
npm start
```

### 3. 切换本地模型（Ollama）

在 `ai-assistant.html` 中将 `API_CONFIG.useLocal` 设为 `true`，并确保本地已运行：
```bash
ollama run qwen3:7b
```
